接著切入main(目的地)，點自己merge，然後push  

 Merge（合併分支）：將一個分支的內容合併到目前分支。  
 1.確認你目前「所在的分支」是你想合併進去的分支（例如 main）。 
 2.點選上方工具列的 Merge。   
 3.在彈出視窗選擇你要合併進來的分支（例如 feature/login）。  
 4.點擊 OK 合併。  
 5.若有衝突，SourceTree 會提示你解決衝突。  
 6.你可以透過內建的合併工具或手動編輯檔案。  
 7.解完衝突後，要記得 Commit merge。
